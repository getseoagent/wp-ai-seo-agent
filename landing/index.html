The problem

Per-page AI buttons don't scale.

Other AI SEO plugins give you a "rewrite" button on a single post. One click, one rewrite, one page. That doesn't help when you have 500 product descriptions written in the wrong tone, or 200 blog posts with stale meta descriptions.

Our approach

Sample-and-extrapolate.

Describe what you want. Agent fetches a sample, proposes 5 diffs, you approve. Then it applies the same style to the remaining N posts in one bulk operation, with full audit history and one-click rollback.

FAQ

Honest answers.

Do I need an Anthropic account?

Yes. The plugin is bring-your-own-key — you create an Anthropic account, generate an API key, and paste it into plugin settings. The key is encrypted at rest in your WordPress database, and forwarded once per chat request to our backend, which forwards it to Anthropic for that single request and does not persist it.

Does it replace my existing SEO plugin?

No. AI SEO Agent writes through your existing plugin's storage (RankMath, Yoast, AIOSEO, or SEOPress), so all your analysis, sitemaps, and schema markup keep working unchanged.

Can I undo a bulk run?

Yes. Every change is recorded in an audit log keyed by job ID. Click Rollback all on any completed bulk job and the original values are restored.

Is my post content sent to a third party?

To rewrite SEO fields, the relevant post title, content excerpts, and existing SEO field values are sent to Anthropic via our backend, using your API key. We do not retain post content beyond the chat session retention window (default 90 days).

Can I self-host the backend?

Yes. The backend ships as a Docker Compose stack or systemd unit. The plugin from WordPress.org is the same in both cases — it points at whichever backend URL you configure.

What's the catch?

No catch. Free tier is genuinely free forever — chat plus all read-only tools. We charge for write tools and bulk operations because that's where the heavy backend cost lives. Cancel any time, no card-on-file lock-in.
